# StatsPreviewCardComponent
A responsive stats card built for the **Frontend Mentor** challenge using **HTML** and **CSS**.  
The project focuses on clean layout, responsive design, and clear visual hierarchy

## Overview
This component displays business statistics in a modern card layout, combining text content with an image and a subtle color overlay.
It is fully responsive and provides a consistent experience across mobile and desktop screens.

## Objectives
- Build a pixel-accurate UI based on a design reference
- Practice responsive layouts using **Flexbox**
- Implement mobile-first CSS workflow
- Apply image overlay effects using CSS
- Improve typography using custom Google Fonts

## Technologies Used
- **HTML5**
- **CSS3**
- Flexbox
- Media Queries
- Google Fonts:
  - **Inter**
  - **Lexend Deca**

## Responsive Behavior
- **Mobile (default)**
  - Vertical card layout
  - Center-aligned text
  - Image displayed at the top

- **Tablet & Desktop**
  - Horizontal layout
  - Image positioned on the right
  - Left-aligned content
  - Stats displayed in a single row

## Features
- Fully responsive design
- Clean and semantic HTML structure
- Mobile-first approach
- Background image with colored overlay
- Well-structured and readable CSS
- Optimized typography and spacing
